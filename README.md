# Análise Preditiva da Seleção Brasileira – Copa do Mundo 2026

Este projeto foi desenvolvido em Power BI com o objetivo de aplicar análise de dados e visual analytics em um cenário real: a projeção de possíveis convocações da Seleção Brasileira e a estimativa de probabilidade de título na Copa do Mundo de 2026.

## Objetivos do Projeto
- Criar um ranking de jogadores com base na probabilidade de convocação
- Projetar um possível time titular
- Analisar a disputa por posição
- Comparar o Brasil com outras seleções
- Estimar a probabilidade de título

## Ferramentas Utilizadas
- Power BI
- DAX
- Modelagem de dados
- Visual Analytics



## Observação
Este projeto representa um modelo preditivo baseado em dados e cenários, não uma previsão oficial ou convocação real.
